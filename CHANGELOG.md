# Changelog

All notable changes to Framewright will be documented in this file.

## 0.1.0 - Initial Open Source Release

- Added visual iframe inspector for selecting, dragging, resizing, and editing text.
- Added stable IDs with `data-fw-id`, `data-block-id`, and `data-frame-id`.
- Added modular architecture layer for component registry, virtual tree, scoped CSS, shadow mapping, prompts, patches, rollback, metrics, and manifest.
- Added dual-layer source editing with HTML DOM AST writeback.
- Added local fast path for deterministic visual edits before AI fallback.
- Added OpenAI-compatible streaming model integration.
- Added architecture tests and architecture audit command.
- Added deployment, security, roadmap, and contribution documentation.
