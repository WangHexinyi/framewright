# Code of Conduct

Framewright uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a respectful and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on the work, not the person.
- Accept responsibility for mistakes and correct them.
- Respect different experience levels and communication styles.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.
- Any behavior that would make the project unsafe for contributors or users.

## Enforcement

Maintainers may remove comments, close issues, or restrict participation when behavior violates this code.
